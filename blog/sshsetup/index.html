Transfer files to Roar Collab from the terminal

Mar 7 2024 · 1 min read
#SSH #server #setup

Overview

I often find that using open OnDemand interactive desktop

  1. Start the SSH Agent
eval `ssh-agent -s`
  1. Generate key 
    ssh-keygen
    When prompted choose a name for the ssh key. Don’t set a passphrase (just click enter).
  2. Copy key to server 
    export PS1="\e[0;35m[\u@\W]\$\e[m"
