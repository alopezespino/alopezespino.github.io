Setup for Developing Projects

by Alejandra López Espino | Mar 9 2024 · 1 min read

Managing environment

using Anaconda

  • On Roar
module load anaconda

Create an environment

  • with a path
conda create --prefix path/to/env
  • with a name
conda create --name env_name

Load environment

  • with a path
conda activate path/to/env
  • with a name
conda activate env_name

Unload environment

conda deactivate

List conda environments

conda info --envs

Erase environment

conda env remove --name myenv

Search for a package

conda search package

To install packages

conda install package

If not available, try

mamba install -c conda-forge package

for more details consult the official Anaconda and Mamba documentation.
